A few notes about the sections above:

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `woo-top-orders-countries` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open the WooCommerce Admin dashboard pages and enable the leaderboard Top countries - orders.
4. Now you can view the leaderboard for the listing the top countries were you sold items.
